日常工作流程

  1. 克隆仓库git clone https://github.com/username/repo.git
  2. 创建并切换分支git checkout -b feature-branch
  3. 修改代码并提交git add . git commit -m "新增功能"
  4. 推送到远程分支git push -u origin feature-branch
  5. 切换回主分支并拉取最新的代码git checkout main git pull
